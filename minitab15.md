# DOE
## 全因子设计
- 常用1-4个因子，具体操作：
  ```
  统计 -> DOE -> 因子 -> 创建因子设计 ->
  2水平因子（默认生成元） -> 因子数 -> 设计 ->
                                            全因子 -> 每个区域的中心点数（中心点实验次数） -> 角点的仿行数（一般为1 即只实验一次 无重复 ） -> 区组数（一般为1，即同一天 同一设备 同一环境之意）
                                         -> 因子（输入名称 设置高低值）
  ```
- 标准化效应的正态图、Pareto图分析、方差、Y的残差图
  ```
  分析因子设计 -> 选择响应 ->
                 项 -> 所选项里添加或删除直到满足需求 ->
                 图形 -> 勾选正态和Pareto、四合一 -> 
                 结果 -> 一般选择系数和方差分析表(C)
  ```
  - 分析
    - 误差自由度 ≥ 4 时才会计算PRESS和R-Sq(预测)
    - 删减模型分析P值，S （残差标准差）应基本不变或略微减小，R-Sq应略微减小， R-Sq(调整)应增大， R-Sq(预测)应增大， PRESS应减小
- 因子图分析
  ```
  因子图 - 勾选主效应图和交互作用图 - 设置(S) 设置(T) - 所选加入所有需要的因子 - 确定
  ```
  - 主要分析交互性
- 等值线/曲面图分析
  - 可视化配置参数
  - 在分析因子设计 -> 项 -> 取消勾选在模型中包括中心点
  - 等值线/曲面图 -> 设置 -> 勾选变量因子 剩下的其余因子会固定(设置里面设置固定值 其余变量虽可设 但由于是变量 所以没用)
- 响应优化器分析
  - 求最优解
  - 在分析因子设计中删除无关因素 **重要！！！** 取消勾选在模型中包括中心点 **重要！！！**
- 最优解分析
  - 分析因子设计 -> 预测 -> 写入需要的变量的值（这里是最优解值,换行输入多组预测）-> 勾选储存的四个 -> 确定
  - 95%置信区间说明,当烘烤温度取95℃ ,烘烤时间取8h,真空值取-100Kpa无限多次重复运行后,我们有95%的把握断言烘烤后正极水份平均值将落入( 42.415,52.119 )之内
  - 95%预测区间说明，当烘烤温度取95℃,烘烤时间取8h,真空值取-100Kpa验证一次实验，我们有95%的把握断言,烘烤后的正极水份都将落入(38.397,56.137)之内。
